Privacy Policy

NetSuite OpenAir Timesheet Filler

This privacy policy applies to the free CSV-based version of NetSuite OpenAir Timesheet Filler.

What the extension processes

The extension processes the CSV data pasted by the user and the contents of the currently open NetSuite OpenAir or SuiteProjects Pro timesheet page in order to scan available projects and tasks, build a timesheet plan, and fill the user’s timesheet.

How the data is used

The data is used only to support the extension’s single purpose: helping the user turn calendar CSV exports into OpenAir timesheet entries.

Storage

The extension stores saved mappings, theme preference, workday settings, and scanned project metadata locally in Chrome extension storage so the workflow is faster the next time the user opens the extension.

Data sharing

The extension does not send user data to external servers, does not sell user data, and does not use analytics, advertising, or tracking scripts.

Remote code

The extension does not execute remote JavaScript or WebAssembly. All executable code is packaged in the extension itself.

Scope

The extension runs only on NetSuite OpenAir and SuiteProjects Pro pages that match the extension manifest permissions.
