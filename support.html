Support For The CSV Workflow

This page is intended for Chrome Web Store support and user questions. If something is not working as expected, start with the setup checklist below before reaching out.

Quick Setup Checklist

  • Open a NetSuite OpenAir or SuiteProjects Pro timesheet page first.
  • Click Scan Page to load projects and tasks from the current week.
  • Paste a CSV in the expected format: Category, Date-Time, Meeting Name.
  • Map each CSV category to a project and task before previewing.

Common Issues

  • If Scan Page fails, confirm you are on the editable timesheet page and not a read-only view.
  • If project tasks do not appear, run Scan Page again after the timesheet loads fully.
  • If rows are flagged during preview, assign the missing project and task before filling.
  • If fill results look wrong, clear existing entries and refill from a fresh scan.

What The Free Version Includes

  • CSV import
  • OpenAir page scanning
  • Project and task mapping
  • Conflict review and preview plan
  • Fill only, Fill + Save, and Fill + Save & Submit
