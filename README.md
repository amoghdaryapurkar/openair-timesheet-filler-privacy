# Website Files

This folder contains a simple static site for the free extension:

- `index.html` — homepage / official site
- `support.html` — support page
- `privacy.html` — privacy policy

## Recommended Chrome Web Store URLs

After deploying these files to GitHub Pages:

- Official URL / Homepage URL: `/index.html` (or the site root)
- Support URL: `/support.html`
- Privacy policy URL: `/privacy.html`

## Before Publishing

- If desired, add screenshots to `index.html` in the "What Review Looks Like" section.
