Privacy Policy for NetSuite OpenAir Timesheet Filler Pro

Effective date: March 18, 2026

NetSuite OpenAir Timesheet Filler Pro is an independent browser extension designed to help users import calendar data, map entries to NetSuite OpenAir or SuiteProjects Pro projects and tasks, review conflicts, and fill timesheets more efficiently.

Information the extension processes

The extension may process the following information in order to perform its core functionality:

  • Content from the currently open NetSuite OpenAir or SuiteProjects Pro timesheet page, including available projects, tasks, and timesheet fields
  • Calendar event data imported by the user from CSV, Google Calendar, or Microsoft Outlook Calendar
  • User-defined mappings, templates, workday settings, and conflict-handling preferences
  • Authentication tokens required for optional Google or Microsoft calendar import

How information is used

The extension uses this information only to support its single purpose: helping the user import calendar data and complete NetSuite OpenAir or SuiteProjects Pro timesheets.

Storage

User settings, mappings, templates, scanned project metadata, and optional authentication tokens are stored locally in the user’s browser using Chrome extension storage.

Data sharing

The extension does not sell user data. The extension does not transfer user data to third parties except as necessary to support optional calendar import features through Google Calendar APIs and Microsoft Graph APIs, or as otherwise required to provide the extension’s core functionality.

Remote code

The extension does not execute remote JavaScript or remote WebAssembly. All executable extension code is packaged with the extension.

Third-party services

If the user chooses to connect Google Calendar or Microsoft Outlook Calendar, the extension communicates with Google and Microsoft authentication and calendar APIs in order to import calendar event data selected by the user.

Your choices

Users may choose not to use optional Google or Microsoft calendar integration and may instead use manual CSV import.
